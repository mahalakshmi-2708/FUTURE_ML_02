## 🎯 Objective
To build a machine learning system that classifies customer support tickets into categories and assigns priority levels.

---

## 📂 Dataset
A customer support ticket dataset containing ticket descriptions, ticket types, and priority levels was used.

---

## ⚙️ Tools & Technologies Used
- Python  
- Pandas  
- NLTK  
- Scikit-learn  
- TF-IDF Vectorizer  

---

## 🔍 Approach
1. Loaded and explored the dataset  
2. Cleaned text data (lowercasing, removing special characters)  
3. Converted text into numerical features using TF-IDF  
4. Trained a classification model (Naive Bayes / Logistic Regression)  
5. Evaluated performance using accuracy and classification report  
6. Tested the model with sample ticket inputs  

---

## 📈 Output
- Predicted ticket categories based on input text  
- Evaluated model performance (~20% accuracy)  
- Displayed classification report
